# 📅 Smart Study Planner

A modern and interactive **Smart Study Planner Web App** built using **HTML, CSS, and Vanilla JavaScript**.
It helps users plan, organize, and track their daily study tasks with a calendar-based interface.

---

👉 Add your deployed link here

## ✨ Features

* 📆 **Calendar View**

  * Monthly calendar layout
  * Navigate between months
  * Add tasks to specific dates

* 📝 **Task Management**

  * Add, edit, and delete tasks
  * Task details: title, description, priority, status
  * Mark tasks as completed

* 🔄 **Drag & Drop**

  * Move tasks easily between sections

* 💾 **Local Storage**

  * Data persists even after refreshing the page

* 📊 **Dashboard & Analytics**

  * Total tasks, completed tasks, pending tasks
  * Progress tracking
  * Chart visualization (Chart.js)

* 🌙 **Dark Mode**

  * Toggle between light and dark themes

* 🔍 **Search & Filter**

  * Search tasks by title
  * Filter by priority or status

* 📱 **Responsive Design**

  * Works on desktop and mobile devices
